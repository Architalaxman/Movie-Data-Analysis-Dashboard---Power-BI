
# 🎬 Movie Data Analysis Dashboard - Power BI

## 📌 Project Overview
This project analyzes a large movie dataset using Power BI to uncover insights about genres, ratings, revenue, and trends.

Due to large file size, dataset and dashboard are hosted externally.

---

## 🎯 Objectives
- Analyze movie trends over time
- Identify top genres and high-rated movies
- Study revenue and popularity patterns
- Explore correlations between ratings and success

---

## 📂 Dataset Information
- Movies dataset (CSV)
- Includes: Title, Genre, Rating, Revenue, Release Year, etc.

---

## 📊 Dashboard Insights
- 🎭 Genre-wise distribution
- ⭐ Rating analysis
- 📅 Year-wise trends
- 💰 Revenue insights
- 🏆 Top performing movies
